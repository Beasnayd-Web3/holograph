Class Bridge

Remarks

Bridge any holographable contracts and assets across blockchains. The contract abstracts all the complexities of making bridge requests and uses a universal interface to bridge any type of holographable assets.

    Parameters

      The address of the hToken contract that wrapped the origin chain native gas token.

    • hTokenRecipient: `0x${string}`

      The exact amount of hToken reward in wei.

    • doNotRevert: boolean
